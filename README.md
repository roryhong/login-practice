## 環境建置

- Node.js v10.15.0
- Express.js v4.17.1
- Express-handlebars v5.3.2
- Nodemon v2.0.7

## 安裝
在終端機輸入指令 Clone 此專案至電腦

    git clone https://github.com/roryhong/login-practice.git

進入專案目錄

    cd login_check

安裝套件

    npm install

加入種子資料

    npm run seed

啟動專案

    npm run dev

出現以下訊息即可進入localhost:3000查看網站

    app is running on http://localhost:3000
